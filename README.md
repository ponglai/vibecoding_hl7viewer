# HL7 Viewer

A modern, zero-dependency Python application for viewing and parsing HL7 messages.

## Features
- **Modern UI**: Clean, responsive 3-pane layout with Light and Dark themes.
- **Instant Parsing**: Paste any HL7 message (`Ctrl+V`) to instantly see its structure.
- **Detailed Inspection**: 
  - **Segments**: View all segments in a list.
  - **Fields**: Inspect extensive field descriptions (PID, OBR, ORC, etc.).
  - **Components**: Drill down into sub-components.
- **Privacy Focused**: Runs entirely offline. No data is sent to any server.

If you prefer to run the Python code directly:
1. Ensure Python 3.x is installed.
2. Run `python main.py`.
